The Problem
Your test data says John Smith.

Every Arabic app needs test data. Random generators give you "John Smith" with a 555-0123 phone number and a 123 Main St address. That is useless for Arabic software. jadwal gives you "محمد بن عبدالله القحطاني" with a real Saudi phone number, a Riyadh address, and a valid national ID format. Region-specific patterns for Gulf, Egypt, Levant, and Maghreb. Offline. No API.

In Action

Commands
5 commands. Zero config.

generate

Generate synthetic records to file. Supports count, region, output format.

sample

Preview a single record in the terminal. Quick way to see the output.

fields

List all available fields with descriptions and examples per region.

regions

Show supported regions with countries, phone formats, and ID patterns.

explain

Learn how jadwal generates data — patterns, sources, methodology.

Output
Three formats. Your choice.

JSONL

One JSON object per line. Streaming-friendly. Default.

CSV

Standard CSV with headers. Spreadsheet-ready.

JSON

Single JSON array. Full dataset in one file.

Get Started
Arabic test data in seconds.
# Install $ pip install jadwal # Generate 100 Gulf records $ jadwal generate --count 100 --region gulf --output data.jsonl Generated 100 records to data.jsonl # Preview an Egyptian sample $ jadwal sample --region egypt { "full_name": "أحمد محمود عبدالرحمن", "phone": "+20 10 1234 5678", "city": "القاهرة", "national_id": "29001011234567" } # List available fields $ jadwal fields # Export as CSV $ jadwal generate --count 50 --region levant --format csv --output levant.csv
